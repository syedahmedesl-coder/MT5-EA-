# Digital Clock with Multiple Time Zones

A responsive web application that displays the current time in different time zones with real-time updates.

## Features

✅ **Real-time Clock** - Updates every second  
✅ **Multiple Time Zones** - Display time in 12+ major time zones  
✅ **12/24 Hour Format** - Toggle between formats  
✅ **Analog & Digital** - Both clock representations  
✅ **Responsive Design** - Works on desktop and mobile  
✅ **Local Storage** - Remember preferred time zones  
✅ **Clean UI** - Modern, minimalist design  

## Files

- **index.html** - HTML structure with time zone selector
- **styles.css** - Responsive styling with animations
- **script.js** - Real-time clock logic and time zone conversion

## How to Use

1. Open `index.html` in your web browser
2. Select time zones from the dropdown menu
3. Toggle between 12/24 hour format using the button
4. Watch the clocks update in real-time
5. Your preferences are saved automatically

## Time Zones Included

- UTC (Coordinated Universal Time)
- EST (Eastern Standard Time)
- CST (Central Standard Time)
- MST (Mountain Standard Time)
- PST (Pacific Standard Time)
- GMT (Greenwich Mean Time)
- CET (Central European Time)
- IST (India Standard Time)
- JST (Japan Standard Time)
- AEST (Australian Eastern Standard Time)
- NZST (New Zealand Standard Time)
- And more...
